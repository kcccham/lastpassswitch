# LastPass Switch

A simple Chrome extension to toggle LastPass on/off with one click.

## Features

- 🎯 One-click toggle for LastPass extension
- 🎨 Beautiful gradient UI
- ⚡ Instant status updates
- 🛡️ Error handling for edge cases

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" (top right)
3. Click "Load unpacked"
4. Select this folder

## Usage

1. Click the extension icon in your browser toolbar
2. Toggle the switch to enable/disable LastPass
3. Status updates instantly

## Development

**Files:**
- `manifest.json` - Extension configuration
- `popup.html` - UI structure
- `popup.css` - Styling
- `popup.js` - Toggle logic

**Required Permission:**
- `management` - To control other extensions
