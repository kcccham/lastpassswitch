# Privacy Policy for LastPass Switch

LastPass Switch is committed to protecting your privacy. This Privacy Policy explains how we handle information in our Chrome Extension.

### 1. Data Collection
LastPass Switch **does not collect, store, or transmit any personal data or user information.** We do not use cookies, tracking scripts, or any third-party analytics.

### 2. Permissions Usage
Our extension requests the following permission for a specific purpose:
* **management**: This is used solely to detect the presence of the LastPass extension and toggle its enabled/disabled status. We do not access any other data from your installed extensions.

### 3. Third-Party Sharing
Since we do not collect any data, we do not share any information with third parties.

### 4. Remote Code
LastPass Switch does not use any remote code (external JavaScript or WebAssembly). All logic is contained within the extension package.

### 5. Contact
If you have any questions, please contact the developer via GitHub issues.
